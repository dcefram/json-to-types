This tool generates typescript types from a JSON object.

I made this tool as there are times a backend API endpoint's response is not documented. It's easy to manually do it on small responses, but there were multiple endpoints that returned a huge payload... which is what cuased me to make this script :D

Btw, this is just plain HTML with a pint of JS. The styles are from PaperCSS.
